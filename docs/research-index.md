# Research Index

**Check this before any web search.** If a topic matches, read the linked research note first — it contains verified findings that may save a round-trip to the web.

## How to Use

- **Reading:** scan the Keywords column for relevant terms. On a match, read the linked feature folder's `06-research-notes.md`.
- **Writing:** after saving new findings to `06-research-notes.md`, add one row here. Keywords should be the terms a future agent would naturally search for.

## Format

| Keywords | Feature folder | Verified | Finding |
|----------|---------------|----------|---------|
| [searchable terms, comma-separated] | docs/plans/YYYY-MM-DD-feature/ | YYYY-MM-DD | [one-line key finding — the most important thing to know] |
